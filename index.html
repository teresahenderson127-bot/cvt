What is the difference between JPG, PNG, and WEBP?

JPG (JPEG) is a lossy format, great for photographs and complex images where a smaller file size is important. PNG is a lossless format that supports transparency, making it ideal for logos and graphics with sharp lines. WEBP is a modern format developed by Google that offers both lossy and lossless compression, often providing smaller file sizes than both JPG and PNG at comparable quality.
